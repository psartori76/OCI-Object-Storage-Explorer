# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows semantic-style app versioning for packaged releases.

## [1.1.0] - 2026-04-01

### Added

- Full multilingual UI support with Portuguese (Brazil), English, and Spanish
- Centralized Apple-native localization resources with pluralization support
- Localized region display names and localized user-facing diagnostics copy
- GitHub Pages-ready project landing page in `docs/`
- Search-oriented README improvements and English-first repository copy

### Changed

- Updated locale-aware formatting for dates, times, and byte counts
- Refined release packaging metadata for version `1.1.0`
- Standardized GitHub-facing documentation and release notes in English

### Fixed

- Added an explicit Close button to the Diagnostics sheet

## [1.0.0] - 2026-03-31

### Added

- Initial public release of OCI Object Storage Explorer for macOS
- Native SwiftUI desktop UI for Oracle Cloud Infrastructure Object Storage
- OCI authentication, bucket exploration, object browsing, transfers, PAR management, and diagnostics
